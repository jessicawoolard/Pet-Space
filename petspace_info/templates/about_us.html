About Us

My name is Jessica Woolard

I wanted to be able to create an application that would allow pet owners a different way of being able to have their pet found. This app will create a custom QR code for your animal and when scanned will pull up a custom web page that has information all about your pet. This page will also have a link so you can be contacted when that QR code is scanned.

I decided to make this website for a final project that I had to make for Carolina Code school.
