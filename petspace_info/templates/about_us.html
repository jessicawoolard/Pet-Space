About


My name is Jessica Woolard.

I decided to make Pet Space as my final project for the Carolina Code School.

I wanted to be able to create an application that would allow pet owners an alternative way of having their pet found. This web application generates a custom QR code for your pet. When scanned it retrieves a custom pet profile that has information all about your pet. This page provides a link so you can be contacted when that QR code is scanned.
